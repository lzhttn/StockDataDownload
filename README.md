# 下载股票指数K线数据<br>
基于Baostock的股票和指数K线数据下载工具<br>
依赖：baostock<br>

## 使用方法<br>
直接运行main.py<br>

## 函数说明<br>
- 下载指数的日K线数据<br>
`down_index(code_list)`<br>

- 下载股票的日K线数据<br>
`down_stocks(code_list)`<br>

- 下载股票的5分钟K线数据<br>
`down_stocks_5min(code_list)`<br>

具体请参考<br>
http://baostock.com/baostock/index.php/A%E8%82%A1K%E7%BA%BF%E6%95%B0%E6%8D%AE<br>
